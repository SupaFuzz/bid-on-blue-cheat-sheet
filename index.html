KNOWN ISSUE - QR CODES

Each item in the silent auction has a QR code for bidding. If the user uses the default operating system camera app on their phone to scan the code, this will take them directly to the item on the event website.

If they have not previously logged in (or are using something like iOS's default private browsing feature) this will require the user to login each time to bid.

Alternately, the mobile bidding app (link above) has a QR code icon which will activate the camera from within the Auctria app, bypassing the login session issue on the website.
